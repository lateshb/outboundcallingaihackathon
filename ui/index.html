Onboarding & Quick-Start Setup Manual

1. Database Schema Deployment

Execute the schema statements located in the supabase_schema.sql file inside your Supabase project's SQL Editor window. Row Level Security (RLS) is disabled for local developer operations.

2. Twilio SIP Termination Setup

Ensure you have configured a Twilio Elastic SIP Trunk:

  • Set up a Termination URI (e.g. avb-test-01.pstn.twilio.com) under Twilio Console.
  • Configure IP Access Control or Credentials List for authentication.
  • Verify the phone number is correctly configured and pointing caller ID details to the trunk.

3. LiveKit Agents Worker Initialization

Launch the Python backend service locally by executing: python server.py.

Launch the background Agent worker process by executing: python agent.py start.
